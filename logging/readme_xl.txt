Инструкция к расписанию в ексель:

Нельзя удалять, вставлять строки.
Можно менять, копировать, удалять содержимое только внутри таблицы (колонки: "скорость", "пуск", "стоп").
Нельзя удалять информацию в других ячейках.
Можно менять расписание в любой момент, но не позднее, чем за 11 минут до заданного действия

"Отобразить"   - показывает текущее расписание по выбранной установке
"Применить"    - сохраняет и применяет набранное расписание
"Очистить"     - удаляет текущее расписание
"Очистить все" - удаляет все текущие расписания

Список установок, для которых уже задано расписание в ячейке F13

Если выполнить выход из системы SAUTER в браузере, расписание больше не будет работать

Резервная копия файла расписания находится здесь.....
