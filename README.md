<p>Наименование проекта: Schedules</p>
<p>Цель проекта: Автоматизизация работы оборудования по расписанию.</p>
<p>Описание:<br>
Оборудование уже работает под контролем стороннего веб-приложения Suter ModuWebVision.<br>
Включение/выключение оборудования производится из приложения в ручном режиме.<br>
В проекте реализуется возможность включения/выключения обрудования по заданному расписанию автоматически.<br>
Расписание задается в файле формата excel (применен встроенный язык VBA).<br>
Выполняется периодическое чтение из этого файла и формирование запроса.<br>
Затем выполеняется отправка запроса на сервер в заданное время.<br>
Предусматривается логгирование работы оборудования с отправкой сообщений об аварии в Telegram.<br>
</p>
