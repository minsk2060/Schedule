<p>Наименование: Schedules</p>
<p>Назначение: Проект автоматизирует работу оборудования по расписанию.</p>

<p>Описание:<br>
Оборудование работает под контролем стороннего веб-приложения Suter ModuWebVision.<br>
Расписание задается в файле формата excel (применен встроенный язык VBA).<br>
Выполняется периодическое чтение из файла и формирование запроса.<br>
Затем выполеняется отправка запроса на сервер при помощи браузера в заданное время.<br>
</p>
<p>Файлы и папки:<br>
adding/    - сторонные файлы для корректной работы проекта на устройстве:<br>
             1251.zip - комплект необходим для корректного отображения русского текста
                        в скриптах VBA.<br>
excel/     - файлы формата excel:<br>
             Расписание.xlsm - пользовательский файл, в котором задается распсиание.<br>   

logging/   - файлы логов формата txt:<br>
             bug_1,bug_2,...- нетиповые ошибки в ходе работы над проектом и методы их решения<br>
             log_scheduling - отправленные на сервер последние действия в пользовательском виде<br>
             alllogs        - отправленные на сервер действия, но за более длительный период<br>
             readme_xl      - краткое пояснение для работы с файлом Расписание.xlsm<br>
             steps_py       - задачи для решения в python<br>
             steps_xl       - задачи для решения в excel<br>

trying/    - пробные сценариии<br>

excels.py  - работа с excel <br>
Scheduls.py- основная логика<br>
plants.py  - словарь с перечнем оборудования и кодов<br>
logs.py    - логгирование<br>
request.py - формирование запросов<br>
