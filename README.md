Наименование: Schedules
Назначение:Проект автоматизирует работу оборудования по расписанию.

Описание:
Оборудование работает под контролем стороннего веб-приложения Suter ModuWebVision.
Расписание задается в файле формата excel (применен встроенный язык VBA).
Выполняется периодическое чтение из файла и формирование запроса.
Затем выполеняется отправка запроса на сервер при помощи браузера в заданное время.

Файлы и папки:
adding/    - сторонные файлы для корректной работы проекта на устройстве:
             1251.zip - комплект необходим для корректного отображения русского текста
                        в скриптах VBA.
excel/     - файлы формата excel:
             Расписание.xlsm - пользовательский файл, в котором задается распсиание.   

logging/   - файлы логов формата txt:
             bug_1,bug_2,...- нетиповые ошибки в ходе работы над проектом и методы их решения
             log_scheduling - отправленные на сервер последние действия в пользовательском виде
             alllogs        - отправленные на сервер действия, но за более длительный период
             readme_xl      - краткое пояснение для работы с файлом Расписание.xlsm
             steps_py       - задачи для решения в python
             steps_xl       - задачи для решения в excel

trying/    - пробные сценариии

excels.py  - работа с excel 
Scheduls.py- основная логика
plants.py  - словарь с перечнем оборудования и кодов
logs.py    - логгирование
